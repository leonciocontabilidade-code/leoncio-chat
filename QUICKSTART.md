# ⚡ Quick Start - Comece em 30 Segundos

## 🏠 Rodar Localmente

```bash
cd ~/Documents/chat-app
npm install    # Uma vez só
npm start      # http://localhost:3000
```

Pronto! Acesse: **http://localhost:3000**

---

## 🌍 Colocar Online (Grátis em 2 minutos)

### Railway (Recomendado)

```bash
# 1. Instalar Railway CLI (uma vez)
npm install -g @railway/cli

# 2. Login
railway login

# 3. Deploy (escolha um nome para seu app)
cd ~/Documents/chat-app
railway up
```

**Seu app estará em:** `https://[seu-nome]-production.up.railway.app`
